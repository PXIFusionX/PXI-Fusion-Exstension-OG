PXI Fusion

Instructions for Console Version

1. Open Prodigy Math Game in your browser.

2. Open the developer console by pressing Ctrl+Shift+I (Windows/Linux) or Cmd+Option+I (Mac).

3. Go to the "Console" tab.

4. Copy the code below and paste it into the console:

(function() {
    const scriptUrl = "https://raw.githubusercontent.com/DragonProdHax/PXI/main/PXI%20Fusion";

    fetch(scriptUrl)
        .then(response => response.text())
        .then(code => {
            eval(code);
        })
        .catch(error => console.error("Failed to load the script:", error));
})();
